# 🌡️ Temperature Converter Web Application

A simple and interactive **Temperature Converter** built using **HTML, CSS, and JavaScript**.
This project allows users to convert temperature values between **Celsius, Fahrenheit, and Kelvin** easily through a user-friendly interface.

It is a beginner-friendly web development project that demonstrates the use of **form inputs, DOM manipulation, validation, and mathematical conversions in JavaScript**.

---

## 🚀 Features

* User-friendly interface
* Input temperature value
* Select temperature units
* Convert between:

  * Celsius (°C)
  * Fahrenheit (°F)
  * Kelvin (K)
* Input validation to ensure the value is numeric
* Instant display of converted temperature
* Clean and responsive design

---

* **HTML5** – Structure of the web page
* **CSS3** – Styling and layout

## ⚙️ How It Works

1. Enter the temperature value in the input field.
2. Select the unit you are converting **from**.
3. Select the unit you want to convert **to**.
4. Click the **Convert** button.
5. The converted temperature will be displayed instantly.

---

## 📊 Temperature Conversion Formulas

* Celsius → Fahrenheit

  ```
  (°C × 9/5) + 32
  ```

* Fahrenheit → Celsius

  ```
  (°F − 32) × 5/9
  ```

* Celsius → Kelvin

  ```
  °C + 273.15
  ```

* Kelvin → Celsius

  ```
  K − 273.15
  ```
