# Personal Portfolio Website

A **Personal Portfolio Website** developed using **HTML, CSS, and JavaScript**.
This project is designed as a beginner-friendly **web development internship project** that demonstrates fundamental front-end development skills.

The portfolio allows users to introduce themselves, showcase their skills, highlight projects, and provide contact information in a clean and responsive layout.

---

## 📌 Project Overview

This portfolio website is created to demonstrate basic web development concepts such as:

* Structuring web pages using **HTML**
* Styling and layout design using **CSS**
* Adding interactivity using **JavaScript**
* Creating a responsive and visually appealing personal portfolio

The website includes several sections to display information about the developer, including skills, projects, resume, and contact details.

---

## 🚀 Features

* Responsive **Navigation Bar**
* **Hero Section** with profile introduction
* **About Me Section**
* **Skills Section** with progress bars
* **Projects Section** to showcase work
* **Resume Download Option**
* **Contact Form**
* **Social Media Links**
* **Smooth Scrolling Navigation**
* **Interactive Buttons using JavaScript**

---

## 🛠 Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout design
* **JavaScript** – Interactivity and dynamic behavior

## ⚙️ How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Double-click **index.html** or open it in any web browser.

No additional installation or dependencies are required.
