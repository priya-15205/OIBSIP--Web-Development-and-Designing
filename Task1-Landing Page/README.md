# Landing Page Project

## 📌 Project Overview

This project is a **simple Landing Page** created as part of a **Web Development Internship task**.
It demonstrates the fundamental concepts of **HTML, CSS, and basic JavaScript** by building a structured and visually appealing webpage.

Landing pages are excellent beginner projects because they help developers understand how to:

* Structure a webpage using **HTML**
* Design layouts and styling using **CSS**
* Add basic interactivity using **JavaScript**

The main goal of this project is to create a **clean, well-aligned layout** while ensuring that elements such as headers, sections, columns, and footers are properly arranged without overlapping.

---

## 🎯 Objectives

* Practice **HTML page structure**
* Use **CSS for layout, alignment, and design**
* Learn how to divide a webpage into **multiple sections**
* Apply **padding, margins, and color palettes**
* Create **columns using Flexbox**
* Implement a **basic JavaScript interaction**

---

## 🧰 Technologies Used

* **HTML5** – Structure of the webpage
* **CSS3** – Styling and layout design
* **JavaScript** – Basic interactivity

## 🖥️ Features

## 📄 Page Sections

The landing page is divided into the following sections:

### 1. Header

Contains the website logo and navigation links such as:

* Home
* Features
* Contact

### 2. Hero Section

A main introduction area that includes:

* A heading
* A short description
* A **"Get Started"** button

### 3. Features Section

Displays **three feature boxes** arranged in columns explaining the key benefits of the page.

### 4. Footer

Contains the project copyright information.

## ⚙️ How to Run the Project

1. Download or clone the project files.
2. Ensure the following files are in the same folder:

   * `index.html`
   * `style.css`
   * `script.js`
3. Open **index.html** in any web browser.

The landing page will load and you can interact with the **Get Started** button.
